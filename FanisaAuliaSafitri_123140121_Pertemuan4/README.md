Program Data Nilai Mahasiswa

Suatu Aplikasi dengan konsol sederhana berbasis Python yang dirancang untuk mengelola, menghitung, maupun menganalisis data nilai mahasiswa.
Program ini memenuhi berbagai kebutuhan dasar seperti menginput data, menghitung nilai akhir, hingga fitur analisis mencari nilai ekstrem dan menghitung rata-rata kelas

Fitur Utama
1. Menghitung nilai akhir secara otomatis
2. Menentukan Grade berdasarkan nilai akhir
3. Fitur menambahkan data mahasiswa baru
4. Menampilkan seluruh data mahasiswa dalam format tabel yang terstruktur
5. Mencari mahasiswa dengan nilai tertinggi atau terendah serta menghitung nilai rata-rata kelas
6. Memfilter data berdasarkan grade tertentu
